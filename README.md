# dadmin - 后台管理系统(模板)
dadmin = django admin
dadmin 是一个基于Python Django框架开发的后台开发模板,用户可以基于此快速开发业务侧.

## 功能特性
- 用户管理
- 角色管理
- 菜单管理
- 权限管理
- 部门管理
- 岗位管理
- 字典管理
- 区域管理  (到乡一级)
- 操作日志管理
- 文件管理
- 图片管理

## 快速开始
### 依赖检查
Python >= 3.6
### 构建
1. pip install requirement.txt
2. python manage.py makemigrations
3. python manage.py migrate
4. python manage.py init
### 运行
python manage.py runserver
